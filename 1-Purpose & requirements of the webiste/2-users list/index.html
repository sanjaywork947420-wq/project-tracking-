Users and Roles

Role 0 – End User

Who?

  • Accessible to every network user with ID
  • Includes all the sailors

What?

  • Can overview their PC details (if PC is linked with their ID)
  • Can use all the web applications end user features
  • Can apply and track complaints

Role 1 – Dept. Head / IT In-Charge

Who?

  • Ship officers
  • Dept. heads
  • IT in charge

What?

  • All features of Role 0
  • Full department item details
  • Putting requirements for AMC staff for any issues

Role 2 – AMC Staffs

Who?

  • AMC staffs

What?

  • All features of Role 1
  • Complaints assigned to them
  • Mark complaints closed, in progress, waiting, or not possible with remarks
  • Report for BR and upload BR certificates

Role 3 – AMC Managers

Who?

  • IT staffs
  • AMC manager civilian

What?

  • All features of Role 1
  • Can assign complaints to AMC staff
  • Create daily or weekly task lists
  • Create accounts for new Role 0 users
  • Can accept/reject complaint requests with remarks

Role 4 – SOIT

Who?

  • IT officer

What?

  • All features of Role 2
  • Can accept/reject complaints and offer BR certificate

Role 5 – Website Manager

What?

  • Can update any details in the code

Role 6 – Developer Account

What?

  • Can change all the codes for the website
